Cara Menjalankan (Offline)

1) Buka file index.html
   - Klik dua kali: index.html
   - Lalu klik tombol "Daftar" atau "Masuk"

2) Daftar akun
   - Isi Nama Lengkap, Nomor HP aktif, dan Skill Utama
   - Setelah daftar otomatis masuk ke dashboard

3) Dashboard
   - Jam bergerak otomatis
   - Absen muka: klik "Aktifkan Kamera" → izinkan kamera → klik "Ambil Foto"
   - Upload portofolio: isi judul, kategori, pilih file (foto/video/dokumen) → simpan
   - Meet Room: klik salah satu link
   - Ruang ngobrol: chat sederhana (tersimpan di browser)
   - Pesan makan: isi pesanan & catatan → tersimpan di riwayat

Catatan penting
- Semua data tersimpan di browser (localStorage). Jika pindah browser/PC, data tidak ikut.
- Kalau ingin hapus semuanya: buka login.html → klik "Reset Data".

Sheet / Export data
- Di halaman dashboard, tekan Ctrl+E untuk download file:
  portfolio-sheet-[nomorHP].json
  Berisi data akun, absen, portofolio, chat, dan pesanan.
